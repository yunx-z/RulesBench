Question:
This question is about the rules of a boardgame called Pax Renaissance (2nd Edition). In the game, can pawn agents participate as attackers in religious wars, conspiracies, and peasant revolts?

Answer:
Pawn agents cannot participate as attackers in religious wars according to the official rules, but they can participate in conspiracies and peasant revolts.

The unofficial rules file which was updated with the living rules seem to indicate something else (i.e., K1a. Attackers. The card's agents [except Bishops*])... see below in red. It sounds like technically you are correct, however I will probably follow these alternative rules for clarity and dynamism...K1. CRUSADE, REFORMATION, OR JIHAD ONE-SHOTIf you launch a Religious War, this creates a BATTLE (B10):a. Attackers. The following are the only attackers in a Religious War and must join the BATTLE:- The card's Agents (except Bishops***)- Bordering Pirates (Believers only)- Ruling Class (Knights & Rooks in that Location, Believers only)- Knights in Adjacent Locations (Believers only, Adjacency includes diagonals, see ADJACENT)....***The original text for K1a included Pawns in the exception for Agent attackers in Religious Wars: "Attackers. The card's Agents (except Bishops or Pawns)...". This results in an exception to the rule that Agents (except for Bishops) are attackers in Battle (see BATTLE, part b), applies to only one card in the game - Zionist State (PREN168X), and makes that card's One-shot rarely useful. Consequently, I removed the exception to reduce rules complexity and to add a bit more potential dynamism to Mamluk (which is often pretty staid).
